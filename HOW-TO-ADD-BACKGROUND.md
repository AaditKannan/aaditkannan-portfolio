# How to Add Background Footage to Resume Page

## Option 1: Video File (Recommended for smooth effect)

### Steps:
1. **Place your video file** in the project root:
   - Name it: `background-video.mp4` (or `.webm`, `.mov`)
   - Or put it in an `assets/` folder: `assets/background-video.mp4`

2. **Supported formats:**
   - `.mp4` (most compatible)
   - `.webm` (smaller file size)
   - `.mov` (if needed)

3. **I'll update the CSS** to use it as a background video

### Video Requirements:
- **Resolution**: 1920x1080 or higher (will scale down)
- **File size**: Try to keep under 10MB for web (or compress it)
- **Duration**: Can be looping video or long clip
- **Format**: H.264 codec for best compatibility

---

## Option 2: Image File

### Steps:
1. **Place your image** in the project root:
   - Name it: `background-image.jpg` (or `.png`, `.webp`)
   - Or put it in an `assets/` folder: `assets/background-image.jpg`

2. **I'll update the CSS** to use it as a background image

### Image Requirements:
- **Resolution**: 1920x1080 or higher
- **File size**: Optimized for web (under 2MB recommended)
- **Format**: JPG (best for photos), PNG (if transparency needed), WebP (smallest)

---
